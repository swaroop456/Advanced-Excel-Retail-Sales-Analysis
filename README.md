# 📊 Advanced Excel Mini Project — Retail Store Sales Analysis

## 📌 Project Overview
This project was completed as part of an Advanced Excel course mini project.
It involves end-to-end data analysis of a Retail Store Sales dataset
containing 12,575 records using Microsoft Excel.

---

## 📁 Files in This Repository

| File | Description |
|------|-------------|
| `retail_store_sales.xlsx` | Complete Excel workbook with all analysis |
| `Adv_Excel_Case_Scenario.pdf` | Project requirements and case scenario |

---

## 🛠️ Tools Used
- Microsoft Excel (Advanced)

---

## 📂 Workbook Structure

| Sheet | Purpose |
|-------|---------|
| `Raw_Data` | Original untouched dataset |
| `Cleaned_Data` | Cleaned data with 15 columns |
| `Data_Analysis` | All formulas and functions |
| `Pivot_Summary` | 4 Pivot Tables and Charts |
| `Advanced_Charts` | Combo, Sparklines, Waterfall |
| `Dashboard` | Interactive KPI Dashboard |
| `Documentation` | Full project documentation |

---

## 🧹 Data Cleaning
- Dataset: 12,575 rows × 11 columns
- Fixed 4,199 blank values in Discount Applied column
- Fixed 1,213 blank values in Item column
- Fixed 604–609 blank values in numeric columns
- Zero duplicates found
- Added 4 helper columns: Month, Year, Month-Year, Sales Tier

---

## 🔢 Formulas & Functions Used
- **Aggregate:** SUM, AVERAGE, COUNTA
- **Conditional:** COUNTIF, SUMIF, SUMIFS
- **Lookup:** XLOOKUP
- **Dynamic Arrays:** UNIQUE, SORT, FILTER, LARGE
- **Logical:** IF, Nested IF
- **Date & Text:** TEXT, YEAR

---

## 📊 Pivot Tables
1. Sales by Category (with Calculated Field)
2. Monthly Sales Trend
3. Payment Method Breakdown
4. Online vs In-Store Transactions

---

## 📈 Key Findings
- **Total Revenue:** $1,552,071 across 3 years (2022–2025)
- **Highest Selling Category:** Butchers — $208,118
- **Most Used Payment:** Cash — $537,710
- **Online vs In-Store:** Online (6,354) slightly exceeded In-Store (6,221)
- **Peak Month:** January 2022 — $52,911.50
- **Avg Order Value:** $123.43 per transaction
